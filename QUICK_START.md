## 🎯 투자 실행 가이드

### 매수 타이밍

**좋은 타이밍**:
- 시장 전체 조정 시 (KOSPI -3% 이상)
- 무관한 악재로 개별 종목 급락 시
- F-Score 재계산 직후 (분기 결산 발표 후)

**피해야 할 타이밍**:
- 급등 직후 (단기 과열)
- 주가가 52주 신고가 근처
- 시장 공황 (시스템 리스크)

### 매수 방법

```
방법 1: 일괄 매수
- 선정한 종목 동시 매수
- 간단하지만 타이밍 리스크

방법 2: 분할 매수 (권장)
- 2-3주에 걸쳐 나눠서 매수
- 평균 매수가 낮춤
- 예: 1주차 50%, 2주차 30%, 3주차 20%

방법 3: 조정 시 매수
- 관심 종목 리스트 작성
- 개별 조정(-10% 이상) 기다려서 매수
```

### 손절/익절 원칙

```
손절 라인 (무조건 실행):
- 대형주: -15%
- 중소형주: -20%
- 턴어라운드: -25%

익절 전략:
- 목표가 없음 (원칙적으로 장기 보유)
- 단, F-Score 하락 시 매도
- 분기마다 재평가 후 판단
```

## 📞 문제 해결

### Q1: 어떤 종목을 골라야 할지 모르겠어요
**A**: 일단 상위 10개(만점 대형주)만 보세요. 삼성바이오로직스, HD현대에너지솔루션부터 시작.

### Q2: 언제 매수해야 하나요?
**A**: 조급하게 서두르지 마세요. 2-3주 관찰하며 조정 시 분할 매수.

### Q3: F-Score만 믿어도 되나요?
**A**: 아니요. 차트, 공시, 뉴스 필수 확인. F-Score는 1차 필터입니다.

### Q4: 얼마나 투자해야 하나요?
**A**:
- 초보자: 100-500만원 (경험 쌓기)
- 경험자: 여유 자금의 30-50%
- 절대 생활비/대출금은 금지

### Q5: 분기별 재계산은 언제 하나요?
**A**:
- 1분기 결산: 5월 중순
- 2분기 결산: 8월 중순
- 3분기 결산: 11월 중순
- 4분기 결산: 3월 중순 (사업보고서)
